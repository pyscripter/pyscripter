***********  This is the 64-bit version of PyScripter. *************
It requires a 64-bit version of Windows. Note that PyScripter will only
work if a 64-bit version of Python is installed.

Before using PyScripter, you must ensure that there is a version of Python 
greater or equal to 2.4 installed on your machine.
If not, you can download one from http://www.python.org/

You can also check if there is a newest version of PyScripter at
http://pyscripter.googlecode.com/

Thanks for your interest in PyScripter.
