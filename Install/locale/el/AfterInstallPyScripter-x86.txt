***********  Αυτή είναι η 32-bit έκδοση του PyScripter. *************
Εάν χρησιμοποιήτε την 64-bit έκδοση των Windows, σημειώστε ότι, το 
PyScripter θα δουλέψει μόνο με εγκατεστημένες 32-bit εκδόσεις της Python.

Πριν χρησιμοποιήσετε το PyScripter, βεβαιωθείτε ότι υπάρχει εγκατεστημένη 
32-bit έκδοση της Python 3.7 ή νεότερη.  Εάν όχι κατεβάστε μια έκδοση της 
Python από το http://www.python.org/.

Επισκεφθείτε τη διεύθυνση https://github.com/pyscripter/pyscripter/wiki 
για πληροφορίες σχετικά με τισ απομακρυσμένες μηχανές Python και άλλα 
θέματα σχετικά με το PyScripter.

Μπορείτε επίσης να ελέγξετε εάν υπάρχει νεότερη έκδοση του PyScripter στο
https://sourceforge.net/projects/pyscripter/files/.

Ευχαριστούμε για το ενδιαφέρον σας για το PyScripter.
