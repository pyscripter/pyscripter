***********  Questa è la versione a 32bit di PyScripter. *************
Se stai usando la versione di Windows a 64bit nota che PyScripter funzionerà solo se è installata una versione di Python a 64bit.

Prima di usare PyScripter, assicurati che sia installata nel computer una versione di Python uguale o superiore alla versione 3.7.
Altrimenti, puoi scaricarla da http://www.python.org/.

Visita https://github.com/pyscripter/pyscripter/wiki per informazioni sul debug remoto e altre info su PyScripter.

Verifica se è disponibile una nuova versione di PyScripter visitando la pagina
https://sourceforge.net/projects/pyscripter/files/.

Grazie per il tuo interesse in PyScripter.
